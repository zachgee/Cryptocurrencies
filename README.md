# Cryptocurrencies

This weeks challenge was all about unsupervised machine learning. The data we will use our machine learning on pertains to a csv with data all about Cryptocurrency. Our assignment consists of 4 components:

- Deliverable 1: Preprocessing the Data for PCA
- Deliverable 2: Reducing Data Dimensions Using PCA
- Deliverable 3: Clustering Cryptocurrencies Using K-means
- Deliverable 4: Visualizing Cryptocurrencies Results

The code for all four deliverables can be found in the crypto_clustering.ipymb file.
